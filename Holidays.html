Holidays

We can only afford to take a trip if the cost is no more than $7,500.

And if the exchange rate falls below 75¢ it will have to be in Australia.

The function trip OK() works out whether we can or can't go on a holiday. You are to test this function.

Use integer numbers for trip cost and exchange rate.
Use true | false for overseas and trip OK.

Notes:
1: Remember testing all boundaries and partitions.
2: Description and coverage are important.
3: Avoid numbers in your description
4: Do not write more than 10 tests. All tests over the 10th won't be considered for making purposes.
